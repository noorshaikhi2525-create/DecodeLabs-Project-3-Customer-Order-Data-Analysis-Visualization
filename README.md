DecodeLabs Project 3 – Customer Order Data Analysis & Visualization
📌 Project Overview
This project involves analyzing customer order data using Python to discover meaningful business insights. The workflow includes data preprocessing, exploratory data analysis (EDA), and visualization to better understand sales performance, customer purchasing trends, and order information.
🛠 Tools & Technologies
Python
Pandas
Matplotlib
Seaborn
VS Code
📂 Dataset Information
The dataset contains order-related information, including:
Order ID
Order Date
Customer ID
Product Details
Quantity
Unit Price
Shipping Address
Payment Method
Order Status
Tracking Number
Cart Items
Coupon Code
Referral Source
Total Price
🧹 Data Preparation
The following preprocessing steps were completed:
Imported the dataset into Python.
Examined the dataset structure and data types.
Detected missing values.
Processed missing values in the Coupon Code column.
Checked for duplicate records.
Converted the Date column into the appropriate datetime format.
🔍 Exploratory Data Analysis
The analysis included:
Statistical summary of the dataset.
Calculation of overall sales.
Identification of the highest-selling products.
Analysis of payment methods.
Examination of order status distribution.
📊 Data Visualization
Several charts were created to present the findings, including:
Bar chart of the top-selling products.
Pie chart of payment methods.
Order status distribution chart.
Sales trend over time.
💡 Project Outcomes
Recognized products with the highest sales volume.
Examined customer payment preferences.
Analyzed order completion status.
Produced a cleaned dataset suitable for further business analysis.
